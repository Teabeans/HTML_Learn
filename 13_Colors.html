ORDERED LISTS

The ordered list tag ('ol') initiates an ordered (numbered) list. It is used the same as the list tag.

  1. Mr. Darling, your cab is here!

TAG ORDER

Note that when using nested tags, the tags should be exited in the order they were entered.
Use a Last-In-First-Out (LIFO) sequence.

COMMENTS

Comment tags can be used to place comments in the source code. Select 'view source' to examine the comments

COLORS

It is recommended that colors and backgrounds by handled by CSS, rather than in .html.
The body tag ('body') contains options (bgcolor="hexadecimalRGB", background="imagename") to set colors and background images.

Text color can be controlled with the body text (body text="hexadecimalRGB")
Link colors, both visited ('vlink') and unvisited ('link') can also be controlled here (body link="hexadecimalRGB")
